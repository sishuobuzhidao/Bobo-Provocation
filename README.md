# Bobo Provocation

This project is about a game that I co-invented with one of my primary school classmates. 
Currently, the project includes a frontend html page and a backend SpringBoot-Maven-Java project.

The game Bobo Provocation is originally a two-player face-to-face game. Players clap their hands twice, and make a gesture of the move it makes.
The game tests the players' resource management skills and predictive, fast-paced analysis of their opponent.

The full rules for the game will not be released here for security and copyright purposes.
