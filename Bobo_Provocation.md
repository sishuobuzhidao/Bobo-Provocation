# 波波挑衅：张XX与钱XX于~2017年原创

双方对弈，每回合同时出招

1）挑衅：免费，不造成伤害，但会被直拳及以上攻击方式击败。可用于兑换直拳、小猩猩冰冻、大猩猩、致命一击、解雇。
2）防御：免费，可抵挡直拳，但会被上勾拳击败。可用于兑换反弹和双层防御
3）左避：免费，可避开直拳，会被左勾拳击败
4）右避：免费，可避开直拳，会被右勾拳击败
5）上勾拳：免费，可击败防御，会被直拳及以上攻击方式击败。可以被反弹（被反弹后对方胜）
6）左勾拳：免费，可击败左避，会被直拳及以上攻击方式击败。不可被反弹
7）右勾拳：免费，可击败右避，会被直拳及以上攻击方式击败。不可被反弹
8）直拳：拥有一个未兑换的挑衅时，可使用直拳。可直接击败挑衅、上勾拳、左勾拳、右勾拳，可以被防御，可以被反弹，可被直拳抵消
9）反弹：拥有一个未兑换防御时，可使用反弹。可反弹直拳、上勾拳和小猩猩。不能防止更高攻击方式
10）小猩猩：拥有三个未兑换的挑衅时，可使用小猩猩。可以被反弹，可被小猩猩抵消。直接击败直拳可击败的所有招式，以及防御、左避和右避
11）双层防御：拥有两个未兑换的防御时，可使用双层防御。可以抵挡直拳、小猩猩、冰冻以及大猩猩。不造成反弹伤害
12）冰冻：累计使用5次挑衅后，可使用冰冻，不需要未兑换的挑衅。使用完冰冻后，前5次挑衅都属于已兑换状态，不能再接着使用直拳/小猩猩，不能再次使用冰冻（除非再次累计满5个挑衅），但前5次挑衅可以继续作为累计计算兑换更高攻击方式。冰冻使对面“冰冻”一回合，即不出任何招式一回合。冰冻只能被冰冻抵消，双层防御防守，或被更高攻击方式击败。
13）大猩猩：累计使用6次挑衅后，可使用大猩猩，不需要未兑换的挑衅（使用完之后的计算方式与冰冻相同。在使用大猩猩之后，若先前没有使用冰冻，则不能使用冰冻，5次计算要重置）。可以直接击败冰冻及以下的招式，只能被双层防御防守，或被大猩猩抵消。
14）致命一击：累计使用7次挑衅后，可使用致命一击（简称“致命”），不需要未兑换的挑衅（使用完之后的计算方式与冰冻相同。在使用致命一击之后，若先前没有使用冰冻和大猩猩，则不能使用，需重新计算累计挑衅数量）。可以直接击败几乎所有招式，无法被防御，只能被致命一击抵消或者被解雇克制。
15）解雇：累计使用10次挑衅后，可使用解雇，不需要未兑换的挑衅（使用完之后的计算方式与冰冻相同，但使用解雇后，若先前未使用冰冻、大猩猩或致命一击，则不能使用，需重新计算累计挑衅数量）。只能被解雇抵消。解雇可使对面强制使用防御5回合（一个上勾拳就能获胜）。


# 攻击方式强度表：
解雇 > 致命一击 > 大猩猩 > 冰冻 > 小猩猩 > 直拳 > 上勾拳、左勾拳、右勾拳
	|	|	|
	可使用双层防御      可使用反弹 可使用防御
